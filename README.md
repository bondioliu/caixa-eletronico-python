# 💰 Caixa Eletrônico em Python

Projeto simples que simula o funcionamento de um caixa eletrônico.

## 🚀 Funcionalidades
- Solicita o valor do saque
- Calcula automaticamente a quantidade de cédulas
- Utiliza as cédulas: 50, 20, 10 e 1

## 🧠 Conceitos utilizados
- Divisão inteira (//)
- Módulo (%)
- Laços de repetição (for)
- Listas em Python

## 📌 Exemplo
Entrada: R$ 286

Saída:
- 5 cédulas de R$ 50
- 1 cédula de R$ 20
- 1 cédula de R$ 10
- 6 cédulas de R$ 1
